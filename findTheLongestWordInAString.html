Find the Longest Word in a String

Return the length of the longest word in the provided sentence.

Your response should be a number.

Required:

  • The length of the longest word in the provided string.

Steps:

  1. Convert the string to an array

  2. Get each element's length

  3. Know the largest number.

  4. Return the element associated with that largest number.

Techniques Used:

  • Spread operator - Spread syntax allows an iterable such as an array expression to be expanded in places where zero or more arguments (for function calls) or elements (for array literals) are expected, or an object expression to be expanded in places where zero or more key-value pairs (for object literals) are expected.

  • array.indexOf()

  • array.map()
